# 🍴 먹어버릴지도

> **성수동 맛집, 더 이상 고민하지 마세요.**

먹어버릴지도는 서울 성수동의 맛집, 카페, 디저트, 술집을 똑똑하게 추천해주는 맛집 큐레이션 서비스입니다.

Powered by 한융원맛집탐험대장 | Beta v2.7

---

## ✨ 주요 기능

### 🎲 랜덤 추천
뭐 먹을지 모르겠을 때, 카테고리를 고르고 돌리기만 하세요. 식사 · 카페 · 디저트 · 술집 중 선택하거나, 메뉴만 뽑기로 오늘의 장르를 결정할 수 있어요.

### 🍜 장르/메뉴별 탐색
한식, 중식, 양식, 일식, 피자, 분식, 돈까스, 국밥, 동남아, 샐러드, 버거, 멕시칸, 탕/찌개, 퓨전, 카페, 디저트 — 16개 카테고리에서 복수 선택 후 추천받기.

### 😊 기분별 추천
로맨틱, 혼밥, 폭주, 치팅데이, 웨이팅싫어, 인스타사진 등 20가지 기분 테마로 지금 내 상태에 딱 맞는 가게를 찾아줍니다.

### 🌤️ 날씨별 추천
실시간 날씨 데이터를 기반으로 비 오는 날엔 국물집, 맑은 날엔 테라스 맛집을 자동 추천합니다.

### 📍 거리 기반 검색
현재 위치에서 100m~2km 반경을 설정하고 가까운 가게부터 탐색하세요.

### 🎯 오늘의 PICK
시간대별로 가장 많이 소비되는 카테고리에서 자동 추천. 점심엔 밥집, 오후엔 카페, 밤엔 술집.

### 💰 더치페이 계산기
N빵, 개별금액, N차 정산까지. 1차 고기, 2차 술, 3차 노래방 — 회차별 멤버가 달라도 최종 정산이 한눈에.

### 🎲 오늘은 니가 쏘냐?
사다리타기, 주사위, 룰렛, 지뢰찾기 — 4가지 미니게임으로 오늘의 결제자를 뽑아보세요.

### ❤️ 여기좋아요
좋아요한 가게를 모아보고, 카테고리별 랜덤 뽑기로 결정 장애를 해결하세요.

### 🔍 퍼지 검색
가게 이름이 정확하지 않아도, 오타가 있어도 찾아주는 스마트 검색.

## 🛠 기술 스택

- **Frontend**: React + Vite
- **Backend**: Vercel Serverless Functions
- **Data**: Kakao Local API (실시간 장소 검색)
- **Weather**: Open-Meteo API
- **Map**: OpenStreetMap (네이버 지도 SDK 전환 예정)
- **Hosting**: Vercel (PWA 지원)
